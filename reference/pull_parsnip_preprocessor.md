# Pulls the Formula from a Fitted Parsnip Model Object

Pulls the Formula from a Fitted Parsnip Model Object

## Usage

``` r
pull_parsnip_preprocessor(object)
```

## Arguments

- object:

  A fitted parsnip model `model_fit` object

## Value

A formula using
[`stats::formula()`](https://rdrr.io/r/stats/formula.html)
